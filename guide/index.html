# Введение

MediaCube Pay - платёжное API для совершения массовых транзакций между сторонними приложениями и MediaCube.ID

# Как это работает

Доступ к API состоит из двух слоёв:

  • OAuth2 (слой аутентификации)
  • Identify Key (слой идентификации/авторизации)
